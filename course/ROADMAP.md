# C++ with xcpp Course Roadmap

## 1) C++ Fundamentals
- Tooling: xcpp/xeus-cling workflow, notebooks, build basics
- Syntax: types, variables, expressions, control flow
- Functions: declarations, overloading, default args
- Memory: stack vs heap, RAII, references, const
- Containers: std::string, std::vector, std::array
- I/O: streams, formatting
- Error handling: exceptions, assertions

## 2) Modern C++ Basics
- Classes/structs, constructors, destructors
- Value vs reference semantics, move semantics
- Templates (intro)
- Namespaces, headers, compilation model
- CMake/compile flags (lightweight overview)

## 3) Data Structures & Algorithms (Simple)
- Big-O intuition
- Arrays, vectors, linked lists
- Stacks, queues, hash maps
- Sorting: bubble, insertion, std::sort
- Searching: linear, binary
- Basic recursion, DFS/BFS (small problems)

## 4) C++ Data Science Stack (Custom Packages)
- NumPy-like arrays (tensor basics)
- Pandas-like DataFrame usage
- CSV loading, column ops, filtering
- Plotting with Matplot++
- Notebook workflows with xcpp

## 5) AI & ML (Intro with LibTorch)
- Tensors, autograd, optimizers
- Linear regression, MLP
- Training loops, metrics, checkpoints
- Model inspection summaries

## 6) NLP + HuggingFace (C++ wrappers)
- Tokenization basics (sentencepiece-lite, tokenizer_decoder)
- Dataset preview/loading
- Model download/load (BERT/DistilBERT)
- Simple inference loops
